# Neraca Mobile Apps

Neraca Mobile Apps is a modern Android application built using **Jetpack Compose**, **Clean Architecture**, and **Modular Design**.

The application is designed to manage authentication flow, session handling, and scalable UI components with a production-ready structure.

---

## ✨ Features

- 📦 Modular Architecture (`app` + `core`)
- 🧠 Clean Architecture (Data, Domain, Presentation)
- 🌐 Ktor HTTP Client
- 🔑 Bearer Token + Refresh Token support
- 💾 DataStore for local persistence
- 🎨 Modern UI with Jetpack Compose
- 🔒 Hilt Dependency Injection
